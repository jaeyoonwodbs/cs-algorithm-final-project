# cs-algorithm-final-project
## Process

- 데이터 삽입
    - 데이터 csv 파일로 불러와서 key에는 날짜, value1/value2에는 각각 지출내역, 카테고리 인덱스 삽입
- 데이터 정렬(option)
- 사용자로부터 날짜 입력받기
- 데이터 탐색(순차, 이진, 트리)
- 입력받은 날짜 범위 내의 지출 내역 출력(내림차순)
- 지출 내역 카테고리화
- 누적 금액 가장 많은 카테고리 출력

<예시>

[입력]

시작 날짜 입력(예:20231101) :
종료 날짜 입력(예:20231130): 

[출력]

-----지출 내역-----
지출: ... 카테고리: ...

-----누적 합-----
카테고리: ... 누적 합: ...

.

.

.
